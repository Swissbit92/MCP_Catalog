## How it all talks

- **Coordinator â†’ Dockerized MCP**: JSON-RPC over **stdio**. We `docker run -i` the server, write JSON lines to its **stdin**, and read responses from **stdout**. `MCPClient` keeps a tiny request/response table and returns results to the HTTP caller.
- **Streamlit â†’ Coordinator**: plain HTTP (`/call`, `/tools`). It stays host-localâ€”no keys, no cloud.
- **MCP servers â†’ services**: inside containers, they use the compose network names:
  - `OLLAMA_BASE=http://ollama:11434`
  - `GRAPHDB_URL=http://graphdb:7200`
  Both are injected by the catalog.
